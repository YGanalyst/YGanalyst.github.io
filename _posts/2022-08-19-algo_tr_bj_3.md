# 개요  

![png](/assets/images/baekjoon.png){: .align-center}{: width="60%" height="60%"}  

source : [백준 10172번](https://www.acmicpc.net/problem/10172)

## 문제  
  
아래 예제와 같이 개를 출력하시오.  

## 입력  

없음.  

## 출력  

개를 출력한다.  

```
|\_/|
|q p|   /}
( 0 )"""\
|"^"`    |
||_/=\\__|
```

<br/>

# 풀이  

## 아이디어  

- 출력문 복사 후 이스케이프 문자 주의해서 삽입  

## 코드  

```python
str_ = '''|\\_/|
|q p|   /}
( 0 )\"\"\"\\
|\"^\"`    |
||_/=\\\\__|'''
print(str_)
```

## 메모  

- 이스케이프 문자 `\`가 필요한 문자열 앞에 삽입한다.  
- 필요한 문자열  
  - `\` , `"`
